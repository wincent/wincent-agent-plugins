# Accessing Atlassian data

Use the Atlassian CLI tool (`acli`) to interact with Atlassian products such as Jira and Confluence.

- `acli help` shows available commands.
- For help on a specific command or subcommand, use `--help`; for example, to see subcommands available for Jira, or to see how to use a specific subcommand like `jira project`, use the following:
  - `acli jira --help`
  - `acli jira project --help`
